# Claude for Lawyers

This is the repository for [HeyCounsel](https://www.heycounsel.com)'s Claude course. It contains the prompts and skills used in each video.

## About the course

This is our foundational course for lawyers who want to learn Claude and start building with AI. No prior experience needed.

It covers the core features of Anthropic's tools, including the Claude desktop app, the Word plugin, and Claude Code. We will provide you with practical examples that you can use in your legal work 
and in the daily operations of your firm. We will also be building tools and workflows that you can use for your practice.
